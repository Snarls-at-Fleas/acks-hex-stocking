Complex Map

The adventurers find a location that is in itself a treasure map. An example might look similar to Stonehenge. It can be difficult to decipher the map, or even to learn that it is a map at all. If deciphered, it leads to a treasure. The treasure is 17 (based on average value) hexes away.

Treasure Type and Average Value

Treasure Type: P (average value: 17000 gp)
